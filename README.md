Данный репозиторий содержит в себе материалы исследований проведенных в рамках подготовки выпускной квалификационной работы по программе магистратуры "Цифровая Урбанистика", реализуемой в Институте дизайна и урбанистики Университета ИТМО, Санкт-Петербург, Россия.
Автор работы - Забавин Иван Евгеньевич. Тема работы - Разработка подхода к планированию размещения мест приложения труда для проектов комплексного развития территории в Санкт-Петербурге.
В числе материалов: текст ВКР, презентация и различные геоданные.
Геоданные редставлены в виде файлов с расширением GeoJSON и GeoPackage и представляют собой: исходные данные, необходимые для расчета, промежуточные результаты построения пространственной модели планирования и итоговые результаты распределения трудовых кластеров.
